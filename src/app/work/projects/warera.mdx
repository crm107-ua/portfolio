## Overview

Warera is a complete web operating system for online gaming communities. It provides a desktop-like web environment with file indexing and an API layer that interconnects multiple games and Discord, so a community can manage data, coordinate strategy and run tools from a single place. Authentication is handled through Discord, so members log in with the account they already use.

## Key Features

- Web OS environment with a desktop-like, windowed UI
- File indexing and management
- API interconnection across multiple games and Discord
- Real-time strategic dashboards for community coordination
- Discord-based single sign-on

## Target Audience

Competitive online gaming communities — clans and alliances that coordinate through Discord and need centralized tooling. The first deployment serves the WarEra España community.

## Business Opportunity

A subscription model per community, plus a white-label / multi-game model where the same web OS can be deployed for other games' communities. Discord-native distribution keeps user acquisition cost low.

## Tech Stack

- **Framework:** Next.js (App Router)
- **API:** tRPC (end-to-end type safety)
- **Auth:** NextAuth with Discord provider
- **Database:** MySQL with Prisma ORM
- **Styling:** Tailwind CSS
