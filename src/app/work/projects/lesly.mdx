## Overview

Multiplatform application for live concert streaming. The first version shipped as a responsive web app with a Laravel frontend; the second version moved the client to Flutter while keeping Laravel as the backend in both iterations. Includes a complete authentication system (sign-up, login, password reset, Facebook OAuth) and an admin dashboard.

## Key Features

- **Auth + social login**: Sign-up, login, password reset, and Facebook OAuth.
- **Admin dashboard**: Content and user management for operators.
- **Responsive UI**: Web-first experience in v1; native mobile in v2.
- **Live streaming**: Concert broadcast workflows across both versions.
